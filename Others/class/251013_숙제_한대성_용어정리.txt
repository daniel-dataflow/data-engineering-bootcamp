- 모집단 : 통계 조사 대상이 되는 집단 전체를 말합니다., 어떤 정보를 얻고자 하는 전체 대상 또는 전체 집합을 의미
- 표본집단 : 모집단으로부터 추출된 모집단의 부분 집합으로 생각 할 수 있음.
- 평균 : 모든 값을 더한 값을 모든 갯수로 나눈 값
- 편차 :  각 데이터가 평균으로부터 떨어져 있는 정도, 편차의 평균은 0
- 표준편차 : 분산에 제곱근을 취한 것
- 분산 : 편차 제곱의 평균값이며 변량들이 퍼져있는 정도를 의미
- 상대도수 : 전체 도수에서 각 계급의 도수가 차지하는 비율
- 누적상대도수 : 특정 계급까지의 모든 상대도수를 더한 값
- 표준화 : 데이터에서 평균을 빼고 표준편차로 나눔.
- 편차값 : 평균이 50, 표준편차가 10이 되도록 정규화한 값
